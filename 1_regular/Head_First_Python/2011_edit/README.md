### 3. 파일과 예외 (에러 처리)

### 4. 영속성 (파일에 데이터 저장하기)
* 메모리에 있는 데이터는 프로그램이 끝나면 휘발되어 사라진다. 
* 이 데이터를 영원히 사용하려면 저장소<sup>storage</sup>에 저장해야 한다. 
* 파이썬 데이터를 저장소에 저장하는 형태로 .pickle을 사용한다. 
    > 데이터 변환 과정을 <b>피클링</b>이라고 부름  
    > 이진화 데이터(binary data)로 바꾸어 저장함
